Fast Synthesis of Power and Temperature Profiles for the Development of Data-Driven Resource Managers

Introduction

The page concerns the following work:

  • I. Ukhov, D. Marculescu, P. Eles, and Z. Peng. Fast synthesis of power and temperature profiles for the development of data-driven resource managers. Tech. rep. Linköping University, Sept. 2017.

Implementation

The implementation is open source. It is written in the Rust programming language, and the corresponding source code is available on GitHub. More concretely, the code is split between three organizations:

  • Stainless Steel, which contains libraries of general interest,
  • Markov Chain, which contains libraries specific to this work, and
  • Learning on Chip, which contains another set of libraries specific to this work.

Stainless Steel

The organization provides libraries for such interrelated disciplines as linear algebra, statistics, probability theory, and numerical analysis.

Markov Chain

The organization provides libraries for simulating timing-, power-, and temperature-related characteristics of computer systems.

The highlights of the organization are:

  • fractal, a multiscale modeling framework, and
  • temperature, a temperature simulator.

Learning on Chip

The organization provides the tools and infrastructures described in the paper.

The highlights of the organization are:

  • recorder, a tool for recording workload patterns,
  • streamer, a tool for generating on-chip data,
  • planner, a tool for constructing floorplans,
  • studio, an infrastructure for recording workload patterns,
  • station, an infrastructure for generating on-chip data,
  • reference-traffic, reference traffic patterns, and
  • reference-workload, reference workload patterns.
