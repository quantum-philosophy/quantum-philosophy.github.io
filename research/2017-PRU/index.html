Fine-Grained Long-Range Prediction of Resource Usage in Computer Clusters

Introduction

The page concerns the following work:

  • I. Ukhov, D. Marculescu, P. Eles, and Z. Peng. Fine-grained long-range prediction of resource usage in computer clusters. Tech. rep. Linköping University, Sept. 2017.

Implementation

The implementation is open source. It is written in the Python programming language using the TensorFlow library, and the corresponding source code is available on GitHub. More concretely, the code is located in the following organization:

  • Learning on Chip.

The highlights of the organization are:

  • google-cluster-data, scripts for processing Google’s cluster data, and
  • google-cluster-prediction, scripts for predicting Google’s cluster data.
