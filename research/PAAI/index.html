Introduction

The page concerns the following work under submission:

  • I. Ukhov, P. Eles, and Z. Peng, “Probabilistic analysis of electronic systems via adaptive hierarchical interpolation.”

Implementation

The implementation is open source. It is written in Go, and the corresponding source code is available on GitHub. Specifically, the code is split between two organizations:

  • Ready Steady, which contains codes of general interest, and
  • Turing Complete, which contains codes specific to simulation of electronic systems.

Ready Steady

The organization deals with such interrelated disciplines as linear algebra, statistics, probability theory, and numerical analysis. In particular, it contains an implementation of the adaptive interpolation algorithm described in the paper.

The highlights of the organization:

  • adapt — the adaptive interpolation algorithm.

Turing Complete

The organization deals with the simulation of timing-, power-, and temperature-related characteristics of multiprocessor platforms executing applications given as directed acyclic task graphs. It also contains the experimental setup described in the paper.

The highlights of the organization:

  • laboratory — the experimental setup.
