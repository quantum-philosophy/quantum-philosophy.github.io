Introduction

This directory contains a set of supplementary materials for the following work:

I. Ukhov, P. Eles, and Z. Peng. “Probabilistic Analysis of Power and Temperature Under Process Variation for Electronic System Design.” IEEE Transactions on Computer-aided Design of Integrated Circuits and Systems, vol. 33, pp. 931–944, June 2004.

Contents

  • Applications — the task graphs of the considered applications (generated by TGFF v3.5 [1]).
  • Platforms — the floorplans of the considered platforms.
  • hotspot.config — the configuration of HotSpot v5.02 [2] used in the experiments.

References

  • [1] R. Dick, D. Rhodes, and W. Wolf. “TGFF: Task Graphs for Free.” In: Proceedings of the Sixth International Workshop on Hardware/Software Co-Design, March 1998, pp. 97–101.
  • [2] K. Skadron, K. Sankaranarayanan, S. Velusamy, D. Tarjan, M. Stan, and W. Huang. “Temperature-aware Microarchitecture: Modeling and Implementation.” ACM Transactions on Architecture and Code Optimization, vol. 1, pp. 94–125, March 2004.
