Introduction

This directory contains a set of supplementary materials for the following publication:

I. Ukhov, M. Villani, P. Eles, and Z. Peng. "Statistical Analysis of Process Variation Based on Indirect Measurements for Electronic System Design." In: Proceedings of the 19th Asia and South Pacific Design Automation Conference (ASP-DAC), January 2014.

Contents

  • Applications — the task graphs of the considered applications (generated by TGFF v3.5 [1]).
  • Platforms — the floorplans of the considered platforms.
  • hotspot.config — the configuration of HotSpot v5.02 [2] used in the experiments.

References

  • [1] R. Dick, D. Rhodes, and W. Wolf. "TGFF: Task Graphs for Free." In: Proceedings of the Sixth International Workshop on Hardware/Software Co-Design, March 1998, pp. 97–101.
  • [2] K. Skadron, K. Sankaranarayanan, S. Velusamy, D. Tarjan, M. Stan, and W. Huang. "Temperature-aware Microarchitecture: Modeling and Implementation." In: ACM Transactions on Architecture and Code Optimization, 1(1):94–125, March 2004.
